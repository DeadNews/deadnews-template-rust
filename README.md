## Installation

Cargo

```sh
cargo install deadnews-template-rust
```

Docker

```sh
docker pull ghcr.io/deadnews/deadnews-template-rust:latest
# or
docker pull ghcr.io/deadnews/deadnews-template-rust:latest-static
```
